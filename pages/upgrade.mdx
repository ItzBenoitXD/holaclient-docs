# 🆙 Updating

To start, choose the way you installed HolaClient.
 
<Tabs items={['Pterodactyl', 'VPS/Terminal']}>
  <Tab>
    You have selected **Pterodactyl**.
    <Steps>
      
    ### Step 1 (Pterodactyl)
    
    Save both `database.sqlite` and `settings.json` in your local PC.
      
    ### Step 2 (Pterodactyl)
      
    Remove all files and change the Version of HolaClient variable in the startup tab to the latest version of HolaClient. If you do not know what the latest version is, click on one of the options below.
    <Tabs items={['1st option', '2nd option (recommended)']}>
       <Tab>
	        You choosed the 1st option.
	        Go to https://github.com/CR072/HolaClient/releases in a another tab and look for a tag with the name **Latest** next to the version. Go back to the server creating page, and put the version in the Version of HolaClient field without including the v in the start.
	     </Tab>
         <Tab>
           You choosed the 2nd option.
		       Go to https://github.com/CR072/HolaClientVersions, and click on **main** next to the number of branches. Click on **View all branches**, and you should see a list of branches. In the **Active branches**, locate the first branch that goes in top of Active branches. It should start with a v in the start. Note the version in your brain and go back to the server creation page. Put the version in the Version of HolaClient field without including the v in the start. Your now done!
         </Tab>
	  </Tabs>
     
    ### Step 3 (Pterodactyl)
    
    Go to your settings tab and reinstall the server. Then, go to the file explorer and check if there's any files. If there isn't, the version you were looking for wasn't found on the versions repository. In this case, please contact our [Discord](https://discord.gg/CvqRH9TrYK).
    
    ### Step 4 (Pterodactyl)
      
    Final step! Remove the existing `settings.json` and `database.sqlite` and upload the `settings.json` and `database.sqlite` files saved on your computer. Then, start your server!
    <Callout type="info" emoji="ℹ️">
      If you wish, you can modify the version setting to the latest version to ensure (and i think) show the version on the admin area.
    </Callout>
      
    ### Step 5 🎉 (Pterodactyl)
      
    Congrats! You successfully upgraded your HolaClient installation! You have reached the end. No, seriously!
    </Steps>
  </Tab>
  <Tab>
    You have selected **VPS/Terminal**.
	<Steps>
      
    ### Step 1 (VPS/Terminal)
    
	Download both `database.sqlite` and `settings.json` from the VPS to your local PC.
	
	### Step 2 (VPS/Terminal)
	
	Execute theses commands to upgrade HolaClient:
	```sh
	pm2 stop holaclient
    rm -rf /var/www/holaclient/
	mkdir /var/www/holaclient && cd /var/www/holaclient/
    git clone --quiet --single-branch --branch VERSION https://github.com/CR072/HolaClientVersions .
	npm i
	```
	
	<Callout type="warning" emoji="⚠️">
       Replace VERSION after --branch with your version of choice STARTING WITH A v! If you do not know the latest version of HolaClient, use this tool below.
    </Callout>
	<Tabs items={['1st option', '2nd option (recommended)']}>
      <Tab>
	    You choosed the 1st option.
	    Go to https://github.com/CR072/HolaClient/releases in a another tab and look for a tag with the name **Latest** next to the version. Then, in the git clone command, replace VERSION with the latest version you noted in your brain including the v at the start.
	  </Tab>
      <Tab>
        You choosed the 2nd option.
		Go to https://github.com/CR072/HolaClientVersions, and click on **main** next to the number of branches. Click on **View all branches**, and you should see a list of branches. In the **Active branches**, locate the first branch that goes in top of Active branches. It should start with a v in the start. Note the version in your brain and replace VERSION with the latest version you noted in your brain including the v at the start. Your now done!
      </Tab>
	</Tabs>
	<Callout type="warning" emoji="⚠️">
      If you encounter any errors, join our Discord in the top right corner that has the Discord logo.
    </Callout>
	
	### Step 3 (VPS/Terminal)

    Remove the existing `settings.json` and `database.json` and upload both `database.sqlite` and `settings.json` to the working directory (/var/www/holaclient/)
	<Callout type="info" emoji="ℹ️">
      If you wish, you can modify the version setting to the latest version to ensure (and i think) show the version on the admin area.
    </Callout>
	
    ### Step 4 (VPS/Terminal)
	
	Final step! Start HolaClient using this command:
	```sh
	pm2 start holaclient
	```
	
	Check if HolaClient is running on the background using `pm2 list` and `pm2 monit`. If it is running in the background, execute theses commands:
	```sh
	pm2 startup
	pm2 save
	```
	<Callout type="warning" emoji="⚠️">
      If you started HolaClient but can't access it, do `pm2 monit`, locate the holaclient process, get the error you're facing and either join our Discord server or submit a GitHub issue in https://github.com/CR072/HolaClient/issues.
    </Callout>
	
	### Step 5 🎉 (VPS/Terminal)
	
	Congrats! You successfully upgraded your HolaClient installation! You have reached the end. No, seriously!
    </Steps>
  </Tab>
</Tabs>
